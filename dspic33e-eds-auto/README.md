## AUTOMATIC EDS USAGE 

## Description:

In this example, two data arrays are defined to be stored in Extended Data Space (EDS) and 
the EDS registers automatically managed by the compiler without requiring the user code
to modify or save/restore the contents of the EDS registers. The code example computes
the Vector Dot Product of the two arrays and stores the result in a variable called 'sum'. 

## Hardware Used

- Explorer 16/32 Development Board (https://www.microchip.com/DM240001-2)
- dsPIC33EP512GM710 PIM (https://www.microchip.com/ma330035) or dsPIC33EP512MU810 PIM (https://www.microchip.com/MA330025-1) or dsPIC33EP256GP506 PIM (https://www.microchip.com/MA330030)	
	
	
## Software Used 

- MPLAB® X IDE v6.00 or newer (https://www.microchip.com/mplabx)
- MPLAB® XC16 v2.00 or newer (https://www.microchip.com/xc)
